# nscanner
Network scanner

[![NPM](https://nodei.co/npm/nscanner.png?compact=true)](https://nodei.co/npm/nscanner/)

## Installation
```bash
npm install nscanner -g
```

## Use
```bash
nscanner
```
OR
```bash
nscanner "ip"
```
